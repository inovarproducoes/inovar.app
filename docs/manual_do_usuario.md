# Manual do Usuário — Inovar App

Bem-vindo ao **Inovar App**, sua plataforma de gestão estratégica para acompanhamento de Ordens de Serviço (OS), alunos e clientes. Este guia ajudará você a navegar e utilizar todas as funcionalidades do sistema com facilidade.

---

## 1. Dashboard Estratégico
O Dashboard é a tela inicial do sistema, onde você tem uma visão consolidada de toda a operação:
- **KPIs Principais:** Total de OS, Ordens Abertas, Finalizadas e Taxa de Conversão.
- **Gráfico de Produtividade:** Acompanhe o volume de OS por mês.
- **OS Recentes:** Lista rápida com as últimas interações no ecossistema.

## 2. Kanban de Ordens de Serviço (OS)
O Kanban é o "coração" operacional do Inovar App. Aqui você gerencia o fluxo de trabalho de ponta a ponta.

### Funcionalidades:
- **Arrastar e Soltar (Drag & Drop):** Mova os cards entre as colunas conforme o progresso da OS.
- **Seletor Rápido de Coluna:** Cada card possui um menu suspenso no rodapé ("Mover...") que permite trocar o status da OS com apenas um clique, ideal para uso rápido no celular ou ajustes precisos.
- **Nova OS:** Clique no botão "+" ou em "Nova OS" no topo ou rodapé de cada coluna para cadastrar uma nova demanda.
- **Busca Inteligente:** Utilize o atalho `Alt + S` para abrir a busca global e localizar qualquer OS, aluno ou cliente em segundos.
- **Arquivamento:** Clique no ícone da lixeira/arquivo no canto inferior direito do card para remover uma OS concluída da visualização ativa.

## 3. Gestão de Clientes e Alunos
Acesse pelos menus laterais para gerenciar os cadastros correspondentes:
- **Alunos:** Cadastro detalhado de estudantes vinculado às OS.
- **Clientes (Parceiros):** Gestão de empresas e contatos responsáveis pelas demandas financeiras e projetos.

## 4. Agente Sophia (Chat de Suporte IA)
No ícone de chat flutuante, você pode falar com a **Sophia**, nossa inteligência artificial treinada para responder dúvidas sobre o sistema e fornecer suporte técnico imediato via WhatsApp.

## 5. Dicas e Atalhos
- **Tema Escuro/Claro:** Alterne as cores do sistema no botão de Sol/Lua no topo.
- **Atalho de Busca:** `Alt + S` foca automaticamente na barra de busca global.
- **Sincronização:** O sistema sincroniza automaticamente com integrações externas (n8n), então novas OS podem surgir automaticamente na coluna inicial conforme vendas são realizadas.
